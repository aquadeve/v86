v86 — Folder Disk (ROOT_DISK/) Example

How it works

v86 can boot from a directory of chunk files on your HTTP server instead of a single large disk image. The directory must contain a manifest.json that describes the total disk size and chunk size; the rest of the chunks are fetched on demand.

To prepare your disk image run:

python3 tools/disk2folder.py --chunk-size 4m my-disk.img ROOT_DISK

Then point v86 at the directory:

hda: { url: "ROOT_DISK/", type: "folder" }

Any static HTTP server works (including python3 -m http.server). The BIOS files must be served from the same origin or CORS must be enabled.
